# Spelling Trainer Pro

An interactive spelling practice application with audio feedback.

## Features
- Multiple grade levels (1-6)
- Text-to-speech pronunciation
- Context sentences
- Hint system
- Scoring and progress tracking
- Dark/light mode
- Neural animation with timer

## How to Use
1. Select a grade level from the sidebar
2. Click "Hear Word" to listen
3. Type your spelling in the input box
4. Check your answer
5. Use hints if needed
6. Advance to next word when correct

## Settings
- **Voice Settings:** Choose from multiple TTS voices
- **Spell Out:** Enable to hear words spelled letter by letter
- **Dark Mode:** Toggle between light and dark themes

## Scoring
- Perfect first try: 10 points
- Correct with hints: 5 points
- Context usage tracked
